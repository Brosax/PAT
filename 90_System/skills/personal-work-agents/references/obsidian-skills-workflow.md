- `.canvas` files or visual canvases: read `90_System\skills\json-canvas\SKILL.md`.

Read only the specific reference files named by the selected skill when they are relevant.

## Modification Standards

- Preserve Obsidian wikilinks and internal note relationships.
- Keep YAML properties valid and minimal.
- Validate `.canvas` as JSON after edits.
- Validate `.base` as YAML after edits.
- Keep generated or edited notes inside the personal vault unless the user explicitly authorizes writing elsewhere.
- Continue to treat shared workgroup folders as read-only except for exact approved file exceptions.
